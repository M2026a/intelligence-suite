Spot Selection Navigator

目的
全国で「どこで食べるか」を決めやすくする静的HTMLツールです。
地方 → 都道府県 → エリア・駅・市 の順で絞り込み、
Google Maps / 食べログ / ホットペッパーグルメの検索導線をすぐ使える構成です。

今回の更新
- 「今日決める」ページを、軸が混ざった分類から
  「シーン別に決める」へ整理しました。
- PCでは2段組み、スマホでは1段表示です。

起動方法
1. Python 3.9以上を用意してください。
2. start_Spot_Selection_Navigator.bat を実行してください。

出力先
output/index.html
