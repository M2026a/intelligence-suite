Travel Search +

目的:
旅行検索をまとめてポン。航空券、鉄道、バス、宿泊、観光、食事、サポート系サイトを静的HTMLからまとめて開ける旅行検索ハブです。

起動方法:
1. start_travel_search_plus.bat を実行
2. output/index.html が開きます
3. 出発地・目的地・日付などを入力して、各カテゴリから検索サイトをまとめて開いてください

出力先:
output/index.html

必要Pythonバージョン:
3.9以上

注意事項:
- output は生成専用です。HTMLを手動編集せず、shared のJSONを編集して再生成してください。
- DBは同梱していません。必要になった場合は実行時に自動生成する前提です。
- tabs.json と sources.json を増やすだけでカテゴリや導線を拡張しやすい構成です。
- 右上の JP/EN と 翻訳風表示 は静的切替です。ページ本文内のラベル説明を切り替えます。
